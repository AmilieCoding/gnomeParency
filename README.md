# gnomeParency
A mirror of a now took down gnome transparency GTK edit.

---
# To Install
- Copy gtk.css to `~/.config/gtk-3.0` and `~.config/gtk-4.0`. (You can append it to the top or bottom of an existing theme. EG - The Everforest Theme)
- Enable BlurMyShell.
- Restart Gnome Shell.
---
